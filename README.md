# M.INC. — Sito statico

Sito statico per M.INC. (Marzorati Incorporation). Contiene le sezioni richieste: Home, DAC, DATANET, Progetti, Chi siamo?.

Come eseguire in locale (Windows PowerShell):

```powershell
# usa lo script start che fa `npx http-server . -p 3000`
npm run start

# poi apri nel browser: http://localhost:3000
```

Nota: lo script utilizza `npx http-server` per un avvio rapido senza installare dipendenze a livello di progetto.

Files principali:
- `index.html` — pagina principale (single-page con sezioni)
- `styles.css` — stili e animazioni
- `scripts.js` — interattività minima
- `package.json` — script `start` per servire il sito
